### 项目介绍
    本项目是针对深大学子快速获取公文通中，对自己有用有利的信息所建立的。

    项目由两部分组成：

    1. 公文通爬虫：利用Python爬取深圳大学公文通，并将信息存储到txt文件中。

    2. 信息分析：利用LLM将用户输入的个人信息与公文通中的信息进行匹配，并给出相应的建议。

    项目的目标是为深大学子提供一个快速、便捷的公文通信息获取方式。

### 环境依赖
建议在conda下创建python3.10.16的虚拟环境。
在虚拟环境中输入：
```
pip install -r requirements.txt
```
### 使用说明

1. 运行爬虫程序： 
    
```
python getRequest.py <账号> <密码>
```

运行后，程序会自动爬取公文通信息，并将信息存储到gongwen.txt文件中。
